Projet : Jeu de devinette de distance

Nom : [Ton nom]

Date : [Date]

1. Introduction

Ce projet consiste à créer un jeu interactif avec un Raspberry Pi Pico. L'utilisateur doit deviner une distance mesurée par un capteur.

2. Description du projet

Le système mesure une distance grâce à un capteur à ultrasons. L'utilisateur entre sa réponse à l'aide d'un clavier.

Ensuite, l'écran affiche :

  • Si la distance réelle est plus grande
  • Si elle est plus petite
  • Ou si la réponse est correcte (± 2 cm)

3. Composants utilisés

Raspberry Pi Pico

C'est le microcontrôleur principal. Il exécute le programme, lit les données du capteur, traite les entrées du clavier et contrôle l'écran.

Breadboard

Elle permet de connecter les composants entre eux sans soudure. Elle est utilisée pour tester et construire le circuit facilement.

Capteur à ultrasons (HC-SR04)

Ce capteur mesure la distance en envoyant une onde sonore et en calculant le temps qu'elle met pour revenir après avoir touché un objet.

Clavier (4x3)

Il permet à l'utilisateur d'entrer sa réponse (la distance qu'il pense correcte).

Matrice LED (MAX7219 Dot Matrix)

Cet écran affiche les messages du jeu, comme "plus", "moins" ou "correct".

4. Fonctionnement

1. Le capteur mesure une distance.
2. L'utilisateur entre une valeur sur le clavier.
3. Le Raspberry Pi compare les deux valeurs.
4. L'écran affiche le résultat.

5. Difficultés rencontrées

- Connexion des composants
- Lecture du clavier
- Précision du capteur
- Affichage sur la matrice LED

6. Conclusion

Ce projet permet d'apprendre à utiliser plusieurs composants électroniques et à créer un système interactif complet.
