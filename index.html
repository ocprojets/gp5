🎮 Devine le Nombre

Projet Raspberry Pi Pico utilisant un clavier, une matrice LED et de la programmation Python pour créer un jeu interactif.

🧠 Concept du Projet

L’ordinateur génère un nombre aléatoire entre 1 et 100. Le joueur doit essayer de deviner ce nombre en utilisant le clavier.

Le panneau LED affiche :

  • ⬆️ Une flèche vers le haut si le nombre entré est trop petit
  • ⬇️ Une flèche vers le bas si le nombre entré est trop grand
  • 😊 Un smiley lorsque le bon nombre est trouvé

⚙️ Fonctionnement du Jeu

🎲

Le Pico choisit un nombre aléatoire entre 1 et 100
⌨️

Le joueur écrit un nombre sur le keypad
🧠

Le programme compare les deux nombres
💡

Le panneau LED affiche un indice
🏆

Le joueur gagne lorsqu’il trouve le bon nombre

🧩 Composants Utilisés

Raspberry Pi Pico

C’est le cerveau du projet. Il exécute le programme Python et contrôle tous les composants.

Clavier 4x3

Le keypad permet au joueur d’entrer ses nombres dans le jeu.

MAX7219 LED Matrix

Le panneau LED affiche les symboles du jeu : flèche vers le haut, flèche vers le bas et smiley.

Breadboard

Elle permet de connecter les composants sans soudure.

💻 Explication du Code

🎲 Nombre Aléatoire

Le programme utilise : 

urandom.randint(1,100)

pour générer un nombre secret.

⌨️ Lecture du Keypad

Le programme détecte les touches pressées sur le clavier pour construire le nombre du joueur.

💡 Affichage LED

Des images 8x8 sont stockées dans des tableaux binaires et envoyées à la matrice LED.

🏆 Conditions de Victoire

Le programme compare la réponse du joueur avec le nombre secret et affiche l’indice correspondant.

🚧 Difficultés Rencontrées

🔌 Connexions

Brancher correctement tous les composants.

⌨️ Lecture du Clavier

Détecter correctement les touches appuyées.

💡 Affichage des Symboles

Créer les flèches et le smiley en format 8x8.

🐞 Débogage

Corriger les bugs du programme et des branchements.

🏁 Conclusion

Ce projet nous a permis d’apprendre :

  • La programmation en Python sur Raspberry Pi Pico
  • L’utilisation d’un keypad
  • Le contrôle d’une matrice LED
  • La logique des jeux interactifs
  • Le travail avec des composants électroniques

Le résultat final est un jeu simple, interactif et amusant.
