🎯 Distance Guess Game

Projet Raspberry Pi Pico

🧠 Concept

Le système mesure une distance. L'utilisateur doit deviner cette distance. L'écran indique si la réponse est plus grande, plus petite ou correcte (±2 cm).

⚙️ Fonctionnement

📡 Mesure

Le capteur à ultrasons mesure la distance.

⌨️ Entrée

L'utilisateur entre une valeur avec le clavier.

🧠 Comparaison

Le Pico compare la réponse avec la vraie distance.

💡 Résultat

L'écran affiche le résultat (plus / moins / correct).

🧩 Composants

Raspberry Pi Pico

Le cerveau du projet. Il exécute le code et contrôle tous les composants.

Breadboard

Permet de connecter les composants sans soudure.

Capteur ultrasons

Mesure la distance grâce aux ondes sonores.

Clavier 4x3

Permet à l'utilisateur d'entrer sa réponse.

Matrice LED MAX7219

Affiche les messages du jeu (plus, moins, correct).

🚧 Difficultés

Connexion des composants
Lecture du clavier
Précision du capteur
Affichage LED
